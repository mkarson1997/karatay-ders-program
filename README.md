# Karatay Ders Programı Oluşturucu (Bilgisayar Programcılığı)

Bu site, **KTO Karatay Üniversitesi – Bilgisayar Programcılığı** öğrencileri için hazırlanmıştır.

## Ne işe yarar?
- Derslerini seçersin (1. sınıf / 2. sınıf / karışık)
- Çakışma varsa uyarı verir (uygunsa otomatik grup değiştirir)
- Programını **PDF** olarak indirirsin

## Kullanım
1) Mod seç (1 / 2 / Karışık)
2) Derslerini işaretle
3) “Programı Göster”
4) “PDF İndir”

> Not: Bu proje sadece Bilgisayar Programcılığı içindir.
